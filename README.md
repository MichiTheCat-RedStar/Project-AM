## Описание

Изначальная идея была в создании анонимного мессенджера, на основе своих протоколов, но сейчас идея проекта в наборе самописных протоколов шифрования, попыток в создание децентрализованных систем, методов безопасности данных и прочее, так же на основе этого мессенджеры, сервера (точнее клиентские части и серверные), иной софт.

Вдохновлено проектами: Tor, XMPP, OMEMO шифрованием, технологиями E2E и P2P, Tox, Brier, Signal так же духовный вклад внёс 2ch.

_Код не пытается быть дружелюбным для пользователей, по большей части он написан для меня самого, либо для обучения._


## Нюансы

**Всё написанное ниже изменить можете именно ВЫ, создав форк или предложив ветку в пул реквесты.**

Код пишется на Linux и тут же компилируется под него, если в файлах найдёте бинарники или отсутствие кроссплатформенности.
